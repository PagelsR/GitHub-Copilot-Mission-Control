# Gino's Gelato Shop

## Unified Demo Prompt

**Prompt:**

```
@workspace /new Create a full-stack web app for an ice cream shop called **“Gino’s Gelato.”** Use **React (TypeScript + Tailwind)** for the frontend and a **.NET backend.**

The app should let users:

* Choose between a cone or cup
* Select up to **three flavors** from a list
* Pick any number of **toppings**
* Add the ice cream to a **cart**

After adding an item, the app should ask if the user wants to view the cart or create another ice cream.

Finally, **improve the design** so the site looks beautiful, modern, and easy to use.
```

## Structured Demo Prompt

**Prompt:**

```
@workspace /new Create a full-stack web app for an ice cream shop called **“Gino’s Gelato.”** Use **React (TypeScript + Tailwind)** for the frontend and a **.NET backend.**

The app should have the following main components:

* **Home Page:** Welcome banner and “Start Your Order” button.
* **IceCreamBuilder Page:**

  * Choose **cone or cup**
  * Select up to **three flavors** from a list
  * Choose any number of **toppings**
  * Add the ice cream to the **cart**
* **Cart Page:** Display selected ice creams, quantities, and total. Include “Checkout” and “Add Another” buttons.

After an order is added, prompt the user to **view the cart or build another ice cream.**

Finally, make the UI **modern, colorful, and mobile-friendly** so it feels like a real gelato shop website.
```

# Brew Haven Café

## Structured Demo Prompt

**Prompt:**

```
@workspace /new Create a full-stack web app for a coffee shop called **“Brew Haven Café.”** Use **React (TypeScript + Tailwind)** for the frontend and a **.NET backend.**

The app should have the following main components:

* **Home Page:** Welcome banner and “Start Your Order” button.
* **CoffeeBuilder Page:**

  * Choose **hot or iced** coffee
  * Select a **base type** (espresso, latte, cappuccino, cold brew, etc.)
  * Pick up to **two flavor shots** (vanilla, caramel, hazelnut, mocha, etc.)
  * Choose **milk options** (whole, oat, almond, soy)
  * Add the drink to the **cart**
* **Cart Page:** Display selected drinks, quantities, and total. Include “Checkout” and “Add Another” buttons.

After a drink is added, prompt the user to **view the cart or build another coffee.**

Finally, make the UI **stylish, cozy, and easy to use**—something that feels like a modern neighborhood café site.
```

# Flight Deck Taproom

## Structured Demo Prompt

**Prompt:**

```
@workspace /new Create a full-stack web app for a brewery taproom called **“Flight Deck Taproom.”** Use **React (TypeScript + Tailwind)** for the frontend and a **.NET backend.**

The app should have these main components:

* **Home Page:** Short intro to the taproom and a “Build Your Flight” button.
* **FlightBuilder Page:**

  * Let the user build a **beer flight** by choosing 3–5 small pours from a list of beers
  * Show beer details like style, ABV, and brief tasting notes
  * Let the user add **food pairings** such as warm pretzels, pizza slices, cheese dip, or nachos
  * Add the customized flight to the **cart**
* **Cart Page:** Display selected flights and pairings with quantities and total. Include “Checkout” and “Build Another Flight” buttons.

After a flight is added, prompt the user to **view the cart or build another flight.**

Make the UI feel like a modern taproom site, with a clean layout that works well on mobile and is easy to use during an in-person visit.
```
